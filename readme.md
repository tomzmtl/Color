# Prism

A color-generation API.

### Endpoints

`api/color` Gets a randomly-generated color in RGB format.

### Color format

Colors are sent as RGB in the following format:

```json
{
  "rgb" : {
    "r" : 45,
    "g" : 234,
    "b" : 89
}
```
